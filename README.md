# Repositório de TCC - Departamento de Estatística

Este repositório contém os códigos e materiais desenvolvidos durante o trabalho de conclusão de curso (TCC) no Departamento de Estatística da Universidade de Brasília (UnB).

## Estrutura do Repositório

- `codigos/`: Scripts e notebooks relacionados ao scraping e análise de dados.
- `dados/`: Dados gerados ou utilizados no projeto, como arquivos CSV.
- `extras/`: Recursos auxiliares, como imagens e logos.
